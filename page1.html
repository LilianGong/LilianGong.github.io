NLP EMOTION DETECTION PROJECT

Course work for Introduction to Machine Learning (10-601)

  1. FEATURE ENGINEERING - based on raw movie review text, I extracted information using bag-of-word model as well as trimmed bag-of word model (only include word that counts less than a threshold t) and turn them into word vector in the form of sparse matrix.
  2. LOGISTIC REGRESSION MODEL - I built up the logistic regression model from scratch, using negative conditional log-likelihood as loss function and stochastic gradient descent to get the parameter elements.
  3.  MODEL EFFICACY - I trained the data in 1200 comments as training set, with the training error of 0.0142; I testified the models in 400 comments as test set, with the testing error of 0.1975.
